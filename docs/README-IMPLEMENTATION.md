## üìö Key Concepts

### Growth Physics Formulas

**Unit Economics**: The foundation of profitable scaling
```
LTV:CAC Ratio = Customer Lifetime Value / Customer Acquisition Cost
Healthy Ratio ‚â• 3:1 (ideally 5:1+)
```

**Churn Impact**: How retention affects LTV
```
True LTV = Monthly Profit / Monthly Churn Rate
Reducing churn by 5% can double LTV
```

**Constraint Identification**: Where to focus efforts
```
If LTV:CAC > 3:1 and margins healthy ‚Üí Constraint is DEMAND
If margins declining with volume ‚Üí Constraint is EFFICIENCY
If close rates high but lead flow low ‚Üí Constraint is DEMAND
If team capacity maxed but demand high ‚Üí Constraint is DELIVERY
```

**Scaling Capacity**: How fast you can grow
```
Revenue = Clients √ó Transaction Frequency √ó Average Ticket Value
Increasing any variable while protecting quality drives growth
```

### Service Business Patterns

**Pattern 1: Recurring Revenue Transforms Economics**
- Hourly billing ‚Üí Retainers (2-3x LTV improvement)
- Projects ‚Üí Subscriptions (predictable, scalable)
- One-time ‚Üí Recurring (reduces CAC payback period)

**Pattern 2: One Constraint Usually Limits Growth**
- Multi-constraint problems are rare
- Fix the primary constraint first
- Don't over-engineer around the wrong bottleneck

**Pattern 3: Unit Economics > Volume**
- Don't scale broken unit economics
- Optimize CAC and LTV before growth acceleration
- Healthy unit economics make scaling capital efficient

**Pattern 4: Quality & Consistency Enable Scaling**
- Quality problems become more visible with growth
- Build systems and standardization first
- People ‚Üí Processes ‚Üí Leverage
